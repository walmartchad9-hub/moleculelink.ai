About this tool

MoleculeLink AI is a pre-synthesis screening tool for medicinal chemists and drug discovery researchers. Paste any SMILES string to receive an instant Drugability Score, Lipinski Rule of Five analysis, and AI-generated feasibility report cross-referencing NIH and FDA databases.

Current limitations: Molecular properties for custom SMILES strings are estimated rather than computed exactly. The four reference compounds (Aspirin, Caffeine, Ibuprofen, Testosterone) pull verified data directly from PubChem. Exact computation via RDKit is coming in Stage 2. The AI feasibility report reasoning is accurate — it is working from estimated inputs for novel structures.

This tool operates exclusively on pre-clinical chemical data. No patient data is processed at any stage.
